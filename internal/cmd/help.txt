Orders:
  spl o l                           List orders
  spl o l --status open             Filter by status
  spl o l --from "2d ago"           Orders since time
  spl o l --email user@test.com     By customer email
  spl o g ORDER_ID                  Order detail (with line items)
  spl o g ORDER_ID --li             Light: id, number, status, total
  spl o q "query"                   Search orders
  spl o mk --body-file order.json   Create order
  spl o up ORDER_ID --note "text"   Update order
  spl o void ORDER_ID               Cancel order
  spl o tg ORDER_ID                 List order tags
  spl o cmt ORDER_ID                List order comments

Products:
  spl p l                           List products
  spl p l --status active           Filter by status
  spl p g PRODUCT_ID                Product detail
  spl p g PRODUCT_ID --li           Light: id, title, status, price
  spl p q "query"                   Search products
  spl p mk --body-file p.json      Create product
  spl p up PRODUCT_ID --title T     Update product
  spl p rm PRODUCT_ID               Delete product
  spl p pub PRODUCT_ID              Publish product
  spl p stk PRODUCT_ID              View stock levels
  spl p var PRODUCT_ID              List variants
  spl p img PRODUCT_ID              List product images

Customers:
  spl cu l                          List customers
  spl cu l --email user@test.com    Filter by email
  spl cu g CUSTOMER_ID              Customer detail
  spl cu g CUSTOMER_ID --li         Light: id, name, email, orders
  spl cu q "query"                  Search customers
  spl cu mk --email e --fn "F"      Create customer
  spl cu up CUSTOMER_ID --note N    Update customer
  spl cu tg CUSTOMER_ID             Customer tags
  spl cu sub CUSTOMER_ID            Subscriptions

Fulfillment & shipping:
  spl ff l --order-id ORDER_ID      List fulfillments
  spl ff g FULFILLMENT_ID           Get fulfillment
  spl shp l --order-id ORDER_ID     List shipments
  spl shp mk --order-id O --tn NUM  Create shipment
  spl sh ex --order-id O            Execute shipment
  spl sh stat --order-id O          Shipment status
  spl sh trk --order-id O           Tracking info

Refunds & returns:
  spl rf l --order-id ORDER_ID      List refunds
  spl rf g REFUND_ID                Get refund
  spl rf mk --order-id O --amt 10   Create refund
  spl ro l                          List return orders
  spl ro g RETURN_ID                Get return order
  spl ro mk --order-id O            Create return
  spl ro rcv RETURN_ID              Mark received
  spl ro cpl RETURN_ID              Complete return

Draft orders:
  spl do l                          List drafts
  spl do g DRAFT_ID                 Get draft detail
  spl do mk --body-file d.json      Create draft
  spl do cpl DRAFT_ID               Complete draft -> order
  spl do si DRAFT_ID                Email invoice

Output formats:
  -o json / -j  Full JSON output
  -o jsonl      Newline-delimited JSON (streaming)
  -o text       Human-readable table (default)
  -q EXPR       JQ expression filter
  --jq EXPR     Alias for --query
  -F FIELDS     Field selection (CSV or preset)
  --items-only  Output only the items array
  --li          Light mode: minimal JSON payload (use with -o json)

Common flags:
  -s STORE      Store profile name (or SHOPLINE_STORE env)
  -l LIMIT      Limit results (sets page size)
  --page N      Page number
  --page-size N Page size
  --sort-by F   Sort by field
  -D            Sort descending
  -y / --yes    Skip confirmations
  --dry-run     Preview without executing
  --no-input    Disable interactive prompts

Exit codes:
  0  Success
  1  General error
  2  Authentication error
  3  Validation error (invalid flags or arguments)
  4  Not found (404)
  5  Rate limited (429)

Environment:
  ~/.openclaw/.env           Auto-loaded when present (existing env vars win)
  SHOPLINE_STORE             Default store profile
  SHOPLINE_ACCESS_TOKEN      API access token
  SHOPLINE_CREDENTIALS_DIR   Override credential storage directory
  CW_CREDENTIALS_DIR         OpenClaw shared credentials root
  SHOPLINE_OUTPUT            Default output format (json|jsonl|text)
  SHOPLINE_ADMIN_TOKEN       Admin API token
  SHOPLINE_ADMIN_MERCHANT_ID Admin merchant ID

Auth:
  spl au login              Browser-based store setup
  spl au l                  List configured stores
  spl au stat               Show current auth status
  spl au rm STORE           Remove store profile

Discovery:
  spl help --json --deep    Full command tree as JSON (for agent discovery)
  spl CMD --help            Detailed help for any command

Use "spl CMD --help" for detailed help on any command.
